This folder contains all the IE416 Group #4 experimenetal data i.e. hand trajectory data, completion time data, error rate data and code files used in data analysis.

Hand Trajectory Data files are labeled in a pattern ('P#_T#_##') for example; 'P1_T1_15' means
P1 - Participant 1
T1 - Participant 2
15 - Button size 15mm
